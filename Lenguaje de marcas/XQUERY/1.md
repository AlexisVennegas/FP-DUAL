
1.- Mostrar los títulos de los libros con la etiqueta "titulo".

    r:  for $x in /bookstore/book/title
        return $x

2.- Mostrar los libros cuyo precio sea menor o igual a 30.

    r:  for $x in /bookstore
        where $x/book/price <= 30
        return $x

3.- Mostrar sólo el título de los libros cuyo precio sea menor o igual a 30.

    r:  for $x in /bookstore
        where $x/book/price <= 30
        return $x/book/title

4.- Mostrar sólo el título sin atributos de los libros cuyo precio sea menor o igual a 30.

    r:  for $x in /bookstore
        where $x/book/price <= 30
        return data($x/book/title)

5.- Mostrar los años de publicación. Etiquetar la salida con "publicacion".

    r: for $x in /bookstore/book/year
        return <publicacion>{string($x)}</publicacion>

6.- Mostrar los libros ordenados primero por "category" y luego por "title" en una sola consulta.

    r:  for $x in /bookstore/book
        order by $x/@categoria, $x/titulo
        return $x

7.- Mostrar cuántos libros hay, y etiquetarlo con "total".

    r:  for $x in /bookstore
        return <total>{count($x/book)}</total>

8.- Mostrar los títulos de los libros y al final una etiqueta con el número total de libros.

    r:  for $x in /bookstore/book
        return (data($x/title)), <total>{count(/bookstore/book)}</total>

9.- Mostrar el precio mínimo y máximo de los libros.

    r:  for $x in /bookstore
        return  
            <resultado>
                <min>{min($x/book/price)}</min>
                <max>{max($x/book/price)}</max>
            </resultado>


10.- Mostrar el título del libro, su precio y su precio con el IVA incluido, cada uno con su propia
etiqueta. Ordénalos por precio con IVA


    r:  for $i in /bookstore/book
        return (
        <resultado>
            <title>{data($i/title)}</title>
            <price>{data($i/price)}</price>
            <iva>{data($i/price) + data($i/price) * .4}</iva>
        </resultado>)

11.- Mostrar la suma total de los precios de los libros con la etiqueta "total".

    r:  for $i in /bookstore
        return 
        <resultado>
            <total>{sum($i/book/price)}</total>
        </resultado>

12.- Mostrar cada uno de los precios de los libros, y al final una nueva etiqueta con la suma de los
    precios.

    r:  for $i in /bookstore/book
        return 
            <resultado>
                <title>{data($i/title)}</title>
                <price>{data($i/price)}</price>
                <sum>{sum(/bookstore/book/price)}</sum>
            </resultado>

13.- Mostrar el título y el número de autores que tiene cada título en etiquetas diferentes

    r:  for $i in /bookstore/book
        return 
        <resultado>
          <title>{data($i/title)}</title>
            <numAuthor>{count(data($i/author))}</numAuthor>
        </resultado>

14.- Mostrar en la misma etiqueta el título y entre paréntesis el número de autores que tiene ese título.

    r:  for $i in /bookstore/book
        return 
            <resultado>
                {data($i/title)}
                ({ count($i/author) })
                </resultado>
