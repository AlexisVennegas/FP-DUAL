
1.- Mostrar los títulos de los libros con la etiqueta "titulo".

    r:  for $x in /bookstore/book/title
        return $x

2.- Mostrar los libros cuyo precio sea menor o igual a 30.

    r:  for $x in /bookstore
        where $x/book/price <= 30
        return $x

3.- Mostrar sólo el título de los libros cuyo precio sea menor o igual a 30.

    r:  for $x in /bookstore
        where $x/book/price <= 30
        return $x/book/title

4.- Mostrar sólo el título sin atributos de los libros cuyo precio sea menor o igual a 30.

    r:  for $x in /bookstore
        where $x/book/price <= 30
        return data($x/book/title)

5.- Mostrar los años de publicación. Etiquetar la salida con "publicacion".

    r: for $x in /bookstore/book/year
        return <publicacion>{string($x)}</publicacion>

6.- Mostrar los libros ordenados primero por "category" y luego por "title" en una sola consulta.

    r:  for $x in /bookstore/book
        order by $x/@categoria, $x/titulo
        return $x

7.- Mostrar cuántos libros hay, y etiquetarlo con "total".

    r:  for $x in /bookstore
        return <total>{count($x/book)}</total>

8.- Mostrar los títulos de los libros y al final una etiqueta con el número total de libros.

    r:  for $x in /bookstore/book
        return (data($x/title)), <total>{count(/bookstore/book)}</total>

9.- Mostrar el precio mínimo y máximo de los libros.

    r:  for $x in /bookstore
        return  
            <resultado>
                <min>{min($x/book/price)}</min>

            </resultado>

EJERCICIOS DE BAILES



1.- Mostrar cada uno de los nombres de los bailes con la etiqueta "losbailes".

r:  for $x in /bailes/baile
    return <losbailes>
    {data($x/nombre)}
    </losbailes>

2.- Mostrar los nombres de los bailes seguidos con el número de plazas entre paréntesis,
ambos dentro de la misma etiqueta "losbailes".

r:for $x in /bailes/baile
    return 
    <losbailes>
        <name>{$x/nombre}</name>
        <place>({data($x/plazas)})</place>
    </losbailes>
    
3. -Mostrar los nombres de los bailes cuyo precio sea mayor de 30.

    r:      for $x in /bailes/baile
            where $x/precio > 30
            return $x/nombre

4.- Mostrar los nombres de los bailes cuyo precio sea mayor de 30 y la moneda "euro".

    r:   for $x in /bailes/baile
        where $x/precio > 30 and $x/precio/@moneda = "euro"
        return $x/nombre

5.- Mostrar los nombres y la fecha de comienzo de los bailes que comiencen el mes de enero
(utiliza para buscarlo la cadena de texto "/1/").

    r:  for $x in /bailes/baile
        where contains($x/comienzo, "/1/")
        return $x
 
6.- Mostrar los nombres de los profesores y la sala en la que dan clase, ordénalos por sala.

    r:   for $x in /bailes/baile
        order by $x/sala
        return 
        <resultado>
            {
                $x/profesor
            }
            {
            $x/sala
            }
        
        </resultado>

7.- Mostrar los nombres de los profesores eliminando los repetidos y acampañar cada nombre
con todas las salas en la que da clase, ordénalos por nombre.

    r:   for $x in distinct-values(bailes/baile/profesor)
        let $sala := /bailes/baile[profesor = $x]/sala
        return 
        <result>
        {
        $x
        }
        {
        $sala
        }
        
        </result>
